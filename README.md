<h1 align="center">My Portfolio ⭐</h1>

## 📝 Project description

<p align="justify">
This repository contains the source code of my personal portfolio, built to showcase my projects, technical skills and experience in software development.

The layout is adapted from the open source template <a href="https://github.com/codewithsadee/vcard-personal-portfolio">vCard - Personal Portfolio</a>, with design and feature customizations built on the Vue ecosystem.

The project includes internationalization (PT/EN) via <code>vue-i18n</code> and lets you export the resume as a PDF (generated in the browser with <code>html2canvas-pro</code> + <code>jspdf</code>).</p>
